synopsis
--------

``diceware`` [`OPTION`]... [`FILE`]...


description
-----------

options
-------

``positional arguments``:

  INFILE
    input wordlist. ``'-'`` will read from stdin.

``optional arguments``:

  -h, --help
    show help message and exit

  -n NUM, --num NUM
    number of words to concatenate. Default 6
